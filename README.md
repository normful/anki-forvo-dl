# anki-forvo-dl
Anki plugin for downloading audio from Forvo

## Getting Started

### Prerequisites

Needs Python and the requests library installed.

```
$ pip install requests
```

A Forvo API key must be located at ~/.forvo.

### Installing
Run the install script and restart Anki.
```
$ ./install
```
